# AI-Based Lung & Colon Cancer Histopathological Image Classification

Deep learning project for multiclass classification of lung and colon histopathological images using ResNet50 transfer learning.

## Features
- ResNet50 Transfer Learning
- 5-Class Histopathology Classification
- Image Preprocessing
- Grad-CAM Explainable AI Visualization
- Confusion Matrix & Evaluation Metrics
- Streamlit Web Application

## Dataset Classes
- Colon Adenocarcinoma
- Normal Colon
- Lung Adenocarcinoma
- Normal Lung
- Lung Squamous Cell Carcinoma

## Results
- Achieved ~93% classification accuracy
- Strong precision, recall, and F1-score performance

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit
- NumPy
- Matplotlib
- Scikit-learn
